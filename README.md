# LOL E-Sport Tools

LOL 电竞经理辅助工具

## 功能

- 选手招聘

## 安装方式

### 方式一

直接在 [Releases Page](https://github.com/HKail/lolet/releases) 中下载对应操作系统的可执行文件并运行.

### 方式二

```
git clone https://github.com/HKail/lolet.git
cd lolet && go run cmd/lolet/main.go
```

## 使用方法

1. 启动项目.
2. 在浏览器中输入 `http://127.0.0.1:8888/home` 访问操作页面.
